1.3 Markdown

Markdown

标题

通过‘#’个数 区分h1-6（标题大小）

# This is an <h1> tag
## This is an <h2> tag
###### This is an <h6> tag

效果：

h1>tag

h2>tag

h6> tag

第二种写法：

Hello {#id}
-----

# Hello {#id}

# Hello # {#id}

效果

Hello

Hello

Hello #

换行

空格和回车都可以

Here's a line for us to start with.

This line is separated from the one above by two newlines, so it will be a *separate paragraph*.

_This will also be italic_
