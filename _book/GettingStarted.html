1.1 入门安装

gitbook有2种模式

1.指令版（cmd指令版：dir\cd\ls）

2.桌面版（有个软件和word一样用）

安装指令版(linux\mac\win(没试过))

npm install gitbook-cli -g

启动图书:进一个页面

gitbook init
